cloudqw-expired

Expired is a Cloudq worker that removes all expired jobs from cloudq

install

npm install cloudqw-expired

use

var config = {
  protocol: 'http',
  host: 'localhost:3000',
  pathname: '/expired',
  interval: 60000
}

exp(config, 'http://localhost:5984/cloudq');
